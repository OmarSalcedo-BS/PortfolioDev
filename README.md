## Portfolio

Proyecto hecho con React, Vite y Tailwind CSS.

### Desarrollo

- `npm install`
- `npm run dev`

### Producción

- `npm run build`
- Subir el repositorio a GitHub
- Conectar el repositorio en Vercel

### Nota de despliegue

El archivo `vercel.json` incluye un rewrite para que `react-router-dom` funcione en rutas como `/proyecto/:id` al refrescar o entrar directo desde Vercel.
